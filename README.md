# Plataforma-Energetica-Brasileira-PEB
Plataforma web centralizada para dados públicos do setor elétrico brasileiro, integrando fontes como ONS, CCEE e ANEEL. Oferece análises  de curtailment, escoamento, PLD, geração renovável, carga do SIN, reservatórios e indicadores do mercado livre, empoderando decisões estratégicas no setor energético.
